# Find My Book
Find My Book este o aplicație desktop pentru Windows care permite cititorilor să găsească rapid cărțile disponibile în bibliotecile digitale și fizice. 
Aplicația oferă linkuri către cărțile online și informații despre biblioteca fizică unde pot fi găsite. 
Scopul principal este de a facilita accesul la informații despre cărți într-un mod intuitiv și rapid, folosind un prototip interactiv creat în Figma.

# Public țintă
Cititori interesați să afle locația cărților
Bibliotecari care doresc să ofere o vizualizare clară a inventarului de cărți

# Funcționalități principale
Gestionarea și vizualizarea cărților
Căutare și filtrare cărți după titlu, autor sau categorie
Link către cărțile disponibile online
Informații despre biblioteca fizică unde se află cartea
Pagini Login / Signup / Reset Password
Chatbot pentru întrebări și sugestii
Pagina profil și Contact

# Design interfață
Top Bar cu navigare rapidă
Carusel pentru evidențierea cărților importante
Pagini intuitive pentru acces la toate funcționalitățile
